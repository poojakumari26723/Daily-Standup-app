# Daily Standup App

## Overview

A browser-only web application for running daily standups with remote teams.

## Requirements Covered

✔ 3–6 participants

✔ Yesterday / Today / Blockers questions

✔ Session summary generation

✔ Shareable text summary

✔ Works entirely in browser

✔ No backend

✔ No database

✔ No server

✔ Pure HTML, CSS and JavaScript

✔ Responsive design

## Design Decisions

### Frontend Only

All data exists only during the session and is processed entirely inside the browser.

### Sequential Flow

Participants answer one-by-one, mimicking a real standup meeting.

### Summary Generation

At completion, the app automatically generates a formatted summary suitable for sharing with absent team members.

### Export Options

Users can:
- Copy summary to clipboard
- Download summary as TXT

## Run

Open:

index.html

in any modern browser.
